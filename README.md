# Michael's reduxApp
#### First React-Redux App

## Install:
```
git clone https://github.com/nivmichael/reduxApp.git
npm install
npm i webpack -g
npm install -g webpack-cli
npm i --save webpack-sources
npm i --save-dev webpack
(install missing dependencies if missing)

```
## Run
```
node server.js
webpack
```
## View
Open localhost:3000
